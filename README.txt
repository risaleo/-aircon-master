GitHub更新方法
1. ZIPを解凍
2. 中の全ファイルをGitHubへ上書き
3. Commit changes
4. 1〜3分待つ
5. Safariで公開URL末尾に ?v=204 を付けて開く
6. ホーム画面にある古いアプリを削除
7. Safariの共有 → ホーム画面に追加

古いアイコンは、ホーム画面から一度削除して再追加しないと変わりません。
